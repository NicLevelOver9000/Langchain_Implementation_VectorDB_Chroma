# Langchain_Implementation_VectorDB_Chroma
Basic RAG Implementation using Langchain and chroma DB.Using gemini-2.5-flash llm model and HuggingFace model sentence-transformers/all-MiniLM-L6-v2 for embedding. ChromaDB for vector database

No credit information or billing account required.

NOTE: ChromaDB does not support Python 3.12 and up as of latest commit.Python 3.11.9 is being used in the virtual environment for this usecase.

#How to run :

-> python -m venv venv #To create a virtual environment

-> venv\Scripts\activate.bat in cmd terminal #To access it

-> pip install -r requirements.txt #To install the required libraries

-> python main.py #To run the program in cmd
